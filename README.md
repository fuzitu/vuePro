##项目前的准备##
- 安装vue npm install vue
- 安装vue-cli npm install --global vue-cli
- vue init webpack my-project
- cd my-project
- npm install
- npm run dev 
通过以上步骤就借助脚手架工具快速的搭起一个vue项目

##利用的技术##
- vue
- vue-router
- axios
- vuex
